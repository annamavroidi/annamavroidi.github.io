Πως να ξεκινήσετε τη διαδικασία αλλαγής της αρνητικής συνομιλίας που έχετε με τον εαυτό σας


Προσπαθήστε τα ακόλουθα:


1.    Όταν αντιληφθείτε ότι είστε αναστατωμένος, προσπαθήστε να θυμηθείτε τις ακριβείς δηλώσεις που κάνατε στον εαυτό σας. Να είστε συγκεκριμένος.

Καταγράψτε αυτές τις δηλώσεις. Ίσως αρχικά να σας φανεί δύσκολο, αλλά συνεχίστε την προσπάθεια.

Τα σχόλια προς τον εαυτό μας γίνονται τόσο αυτόματα ώστε μπορεί στην αρχή να δυσκολευτείτε να τα θυμηθείτε ή να τα αναγνωρίσετε όπως ακριβώς είναι.

 

2.    Κοιτάξτε τη λίστα με τις δηλώσεις που κάνατε στον εαυτό σας. Εφόσον ήσαστε αναστατωμένοι όταν αρχίσατε να τις αναλύετε, θα είναι αρνητικές. Αν οι δηλώσεις που έχετε καταγράψει δεν είναι αρνητικές, τότε δεν τις έχετε εντοπίσει ακόμα. Συνεχίστε να σκέφτεστε. Θυμηθείτε ότι η «επίγνωση» είναι το πρώτο και αποφασιστικότερο βήμα για την απόκτηση κάθε νέας ικανότητας - μπορεί επίσης να χρειαστεί πολύ χρόνο για να αναπτυχθεί.

 

3.    Όταν μπορέσετε να εντοπίσετε μία ή περισσότερες αρνητικές δηλώσεις προς τον εαυτό σας, αναρωτηθείτε αν είναι ακριβείς. Το πιθανότερο είναι ότι ολόκληρη η δήλωση ή ένα μέρος της, δεν είναι. Αν δεν μπορείτε να αναγνωρίσετε τα συγκεκριμένα μέρη, ζητήστε βοήθεια από ένα φίλο σας ή μιλήστε με το θεραπευτή σας γι’ αυτές τις δηλώσεις.

 

4.    Αφού καθορίσετε ποια τμήματα των δηλώσεων σας δεν ανταποκρίνονται στην πραγματικότητα, αντικαταστήστε τα με άλλα που πράγματι ισχύουν. Επαναλάβετε τις καινούριες προτάσεις στον εαυτό σας. Αν κάνετε αυτή την άσκηση σωστά, θα αρχίσετε τελικά να νιώθετε καλύτερα ως προς τη συγκεκριμένη κατάσταση αλλά και ως προς τον εαυτό σας.

 

5.    Υποσχεθείτε στον εαυτό σας ότι θα σταματήσετε να κάνετε δηλώσεις που δεν ανταποκρίνονται στην πραγματικότητα.

 

Έχετε υπ’ όψιν σας ότι δεν πρόκειται για εύκολη άσκηση ούτε για εύκολη διαδικασία. Από τη στιγμή που σκέφτεστε με τον τρόπο που σκέφτεστε για όλη σχεδόν τη ζωή σας, θα είναι δύσκολο να αλλάξετε τις συνήθειες σας. Παρ’ όλα αυτά, μπορείτε να ξεπεράσετε τον αρνητικό και ανακριβή τρόπο με τον οποίο βλέπετε τα πράγματα, αρκεί να είστε αποφασισμένοι να το κάνετε.
