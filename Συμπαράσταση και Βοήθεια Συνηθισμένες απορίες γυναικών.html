Συμπαράσταση και Βοήθεια Συνηθισμένες απορίες γυναικών

 

1η Ερώτηση

Γιατί θα έπρεπε να ρωτάω τον σύντροφο μου αν θέλει να με βοηθήσει, αφού εγώ δεν απαιτώ από εκείνον να με ρωτάει;

Απάντηση

Οι άντρες είναι διαφορετικοί. Αν αποδεχτείτε τις διαφορές του συντρόφου σας, θα σας προσφέρει αυτά που χρειάζεστε. Αντίθετα, αν προσπαθήσετε να τον αλλάξετε, θα σας αντισταθεί με πείσμα.

Αν και δεν σας είναι εύκολο να ζητάτε αυτό που χρειάζεστε, θα πρέπει να μάθετε να το κάνετε.

Όταν ο σύντροφος σας αισθανθεί ότι τον εκτιμάτε και τον αγαπάτε, σταδιακά θα είναι περισσότερο πρόθυμος να σας βοηθήσει χωρίς να του το ζητάτε.

 

2η Ερώτηση

Γιατί να τον ευγνωμονώ γι’ αυτά που κάνει αφού εγώ κάνω περισσότερα;

Απάντηση

Οι άντρες δίνουν λιγότερα όταν νιώθουν ότι δεν τους εκτιμούν. Προκειμένου ο σύντροφος σας να προσφέρει περισσότερα, θα πρέπει να λάβει περισσότερη εκτίμηση. Οι άντρες κινητοποιούνται από την εκτίμηση των συντρόφων τους.

Εάν εσείς προσφέρετε ήδη περισσότερα, είναι δύσκολο να εκτιμήσετε τη δική του προσφορά. Σε αυτή την περίπτωση, αρχίστε σταδιακά να δίνετε λιγότερα ώστε να μπορέσετε να εκτιμήσετε αυτά που σας δίνει εκείνος. Με αυτό τον τρόπο, όχι μόνο του δείχνετε την αγάπη σας, αλλά παίρνετε και τη συμπαράσταση που επιθυμείτε και σας αξίζει.

 

3η Ερώτηση

Αν πρέπει να ζητήσω την βοήθεια του, ίσως πιστέψει ότι μου κάνει χάρη.

Απάντηση

Έτσι θα πρέπει να αισθανθεί. Ένα δώρο αγάπης είναι χάρη.

Όταν ο άντρας αισθάνεται ότι σας κάνει χάρη, σας δίνει ένα κομμάτι της καρδιάς του. Αντιθέτως, αν του πείτε ότι είναι υποχρέωση του να προσφέρει, τότε η καρδιά του κλείνει και σας δίνει πολύ λιγότερα.

 

4η Ερώτηση

Αν με αγαπά, θα πρέπει από μόνος του να μου προσφέρει τη βοήθεια του. Δεν θα πρέπει να τη ζητήσω εγώ.

Απάντηση

Ο άντρας περιμένει να ζητήσετε εσείς την βοήθεια του. Αν δεχτείτε ότι ο σύντροφος σας, ως άντρας, είναι διαφορετικός από εσάς, θα είναι πιο πρόθυμος να σας βοηθήσει και, σταδιακά, θα αρχίσει να σας προσφέρει από μόνος του την υποστήριξη και τη συμπαράσταση του.

 

5η Ερώτηση

Αν αρχίσω να του ζητάω πράγματα, εκείνος μπορεί να νομίσει ότι δεν προσφέρω όσα ο ίδιος. Φοβάμαι ότι θα σκεφτεί πως δεν πρέπει να μου δώσει τίποτε άλλο!

Απάντηση

Ο άντρας αισθάνεται περισσότερο γενναιόδωρος, όταν ξέρει ότι δεν είναι υποχρεωμένος να προσφέρει στη σύντροφο του. Επίσης, όταν ο άντρας ακούει μια γυναίκα να ζητάει, με σωστό τρόπο, την βοήθεια του, αυτό που ακούει είναι ότι η σύντροφος του αισθάνεται πως έχει το δικαίωμα να πάρει αυτή τη βοήθεια. Δεν υποθέτει ότι εκείνη δίνει περισσότερα ή τουλάχιστον το ίδιο με εκείνον και γι’ αυτό αισθάνεται άνετα να ζητήσει τη βοήθεια του.

 

6η Ερώτηση

Όταν ζητάω βοήθεια, φοβάμαι να είμαι σύντομη. Θέλω να εξηγήσω γιατί τη χρειάζομαι. Δεν θέλω να νομίσει ότι είμαι απαιτητική.

Απάντηση

Όταν ο άντρας ακούει την έκκληση για βοήθεια της συντρόφου του, είναι σίγουρος ότι έχει τους λόγους της που ζητάει τη βοήθεια του. Αν του εξηγήσει αναλυτικά γιατί πρέπει να ικανοποιήσει την επιθυμία της, εκείνος αισθάνεται ότι δεν μπορεί να πει όχι, νιώθει ότι η σύντροφος του θέλει να τον χειριστεί ή τον θεωρεί δεδομένο. Αν θέλει να μάθει κάτι επιπλέον, θα ρωτήσει ο ίδιος την σύντροφο του. Τότε, εκείνη μπορεί να του εξηγήσει τους λόγους της. Ακόμα και αν ο σύντροφος την ρωτήσει, εκείνη θα πρέπει να προσέξει να μην αναλύσει κάθε λεπτομέρεια. Να του πει έναν ή το πολύ δύο λόγους. Αν θέλει να μάθει και άλλα, θα το πει ο ίδιος.
