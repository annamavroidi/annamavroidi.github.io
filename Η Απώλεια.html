Η Απώλεια

Η απώλεια είναι η ανθρώπινη εμπειρία που μας φέρνει σε επαφή με ότι πιο σημαντικό υπάρχει στη ζωή μας. Ο συμβιβασμός με το θάνατο είναι η πιο δύσκολη από τις εμπειρίες που αντιμετωπίζουμε στη ζωή.

Η εμπειρία της επαφής με τον θάνατο μπορεί να αλλάξει ολοκληρωτικά τη ζωή μας, να μας προκαλέσει να τη γευτούμε με όλες μας τις αισθήσεις και να προσδιορίσουμε πιο ξεκάθαρα τις προτεραιότητες μας.

Οι πρόωροι θάνατοι αφομοιώνονται πάρα πολύ δύσκολα. Οι τύψεις για κάποια μισοτελειωμένη σχέση μπορεί να κατατρέχουν ένα άτομο για όλη του τη ζωή και όταν οι ανοιχτοί λογαριασμοί δεν τακτοποιούνται, αφήνουν πίσω τους ένα μεγάλο κενό.

Εξετάζοντας τις συνέπειες από τις απώλειες που έχουν οι οικογένειες στις διάφορες γενιές τους, μπορούμε να μάθουμε πολλά για τον τρόπο με τον οποίο λειτουργούν οι οικογένειες, για το τι συμβαίνει όταν κολλάνε κάπου και για το πώς μπορούν να αλλάξουν τα πρότυπα λειτουργίας τους.

Μια απώλεια μπορεί να δυναμώσει αυτούς που μένουν πίσω, φέρνοντας στην επιφάνεια τις δημιουργικές τους ικανότητες και εξωθώντας τους να επιτύχουν πράγματα, ή αντίθετα, μπορεί να αφήσει πίσω της μια καταστροφική κληρονομιά που θα γίνει ακόμα πιο καταστροφική αν δεν αντιμετωπιστεί. Συχνά, χωρίς να το ξέρουμε, αντιμετωπίζουμε το θέμα της απώλειας σύμφωνα με τα πρότυπα που έχουν θεσπίσει οι προηγούμενες γενιές.

Οι εικόνες που κουβαλάμε μέσα μας από τις κηδείες στις οποίες έχουμε παραστεί και τους νεκρούς που έχουμε δει αποτελούν σημαντικό κομμάτι της κληρονομιάς και της ταυτότητας μας.

Ο καλύτερος δρόμος για να προσεγγίσουμε τις βασικές αξίες και αρχές καθώς και τις αδυναμίες της οικογένειας μας είναι να δούμε με ποιόν τρόπο αντιμετωπίζουν το θάνατο.

Λίγο πολύ, όλες οι οικογένειες έχουν σημαδευτεί από κάποια απώλεια. Είναι γενικά αποδεκτό ότι η περίοδος που χαρακτηρίζεται από μια απώλεια είναι μια ευαίσθητη περίοδος κατά την οποία τα μέλη της οικογένειας θα πρέπει να ασχοληθούν το ένα με το άλλο με πιο προσωπικό τρόπο. Αυτό μπορεί να αποβεί εξαιρετικά δύσκολο για οικογένειες που τα μέλη τους δεν έχουν τόσο στενή σχέση. Για παράδειγμα, δύο αδέλφια που ζουν χωριστά για πολλά χρόνια μπορεί να δυσκολευτούν πολύ όταν ξαφνικά θα χρειαστεί να ζήσουν μαζί τη συγκλονιστική εμπειρία της απώλειας ενός κοινού αγαπημένου προσώπου. Αυτή η εμπειρία μπορεί είτε να ξαναενώσει την οικογένεια είτε να ξαναφέρει στην επιφάνεια παλιές συγκρούσεις. Ένας θάνατος μπορεί να δώσει στην οικογένεια την ευκαιρία να εστιάσει ξανά σε κάποιους ουσιαστικούς δεσμούς, να ανοίξει ξανά σχέσεις που είχαν κλείσει και να τις προσδιορίσει με νέο τρόπο ή να διακινδυνεύσει να πει πράγματα που δεν είχαν ειπωθεί. Ωστόσο, αυτά τα ανοίγματα μπορεί να δυναμώσουν τον πόνο που έχουν αφήσει παλιότερες πληγές. Όπως συμβαίνει και με άλλες σημαντικές οικογενειακές εμπειρίες, όταν τα πράγματα δεν βελτιώνονται, συνήθως χειροτερεύουν.

Όταν μιλάμε για απώλεια δεν αναφερόμαστε μόνο στο θάνατο. Απώλεια συνιστά επίσης ένας χωρισμός ή ένα διαζύγιο, μια χρόνια ασθένεια, η έλλειψη δουλειάς ή κατοικίας, ή μια αναπηρία: απώλεια των ονείρων και των προσδοκιών μας.

Στη ζωή, κάθε αλλαγή απαιτεί απώλεια. Είμαστε υποχρεωμένοι να αφήσουμε κάποιες συγκεκριμένες σχέσεις, σχέδια και δυνατότητες για να αποκτήσουμε κάποια άλλα.

Όλες οι απώλειες απαιτούν το πένθος, με το οποίο αναγνωρίζουμε τι έχουμε αφήσει και μεταμορφώνουμε την εμπειρία σε μια διαδικασία με την οποία παίρνουμε αυτό που είναι σημαντικό για μας και μας επιτρέπει να συνεχίσουμε.

Το πένθος είναι δείγμα υγείας. Όταν οι οικογένειες δεν πενθούν επαρκώς για τις απώλειες τους, δεν μπορούν να προχωρήσουν στη ζωή. Αντίθετα, τα συναισθήματα κινούνται υπόγεια. Κάθε μέλος της οικογένειας μπορεί να αρχίσει να κατηγορεί τον εαυτό του ή κάποιο άλλο μέλος για ένα θάνατο. Τα μέλη της οικογένειας μπορεί να προσπαθήσουν να βάλουν κάποιον στη θέση του αντικαταστάτη του προσώπου που έχει χαθεί και να επιχειρήσουν να ζήσουν μαζί του τη σχέση που έχουν χάσει.

Αυτό που καθίσταται προβληματικό δεν είναι ο θάνατος αυτός καθεαυτός, αλλά αυτό που συμβαίνει όταν οι οικογένειες προσπαθούν να αποφύγουν την εμπειρία. Μια οικογένεια μπορεί να προσαρμοστεί ακόμα και στην πιο τραυματική απώλεια. Εάν όμως δεν μπορέσει να αναγνωρίσει την απώλεια και να οργανωθεί ξανά, τότε ακινητοποιείται.
