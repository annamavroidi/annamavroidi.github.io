Όταν το ΟΧΙ δεν μπορεί να ειπωθεί

Γιατί δυσκολεύεστε να πείτε ΟΧΙ;

Δεν μπορείτε να πείτε ΟΧΙ διότι:

·         Φοβάστε πως θα φανείτε αγενής.

·         Φοβάστε τη σύγκρουση.

·         Δεν θέλετε να απομονωθείτε από το σύνολο (οικογένεια, παρέα, αθλητική ομάδα κτλ) και έτσι συμφωνείτε με την συλλογική απόφαση ακόμα και όταν έχετε αντίθετη άποψη.

·         Σας αρέσει να βοηθάτε τους άλλους και να τους είστε χρήσιμος είτε επειδή νιώθετε ηθική ικανοποίηση κάθε φορά που ακούτε «ευχαριστώ» είτε γιατί ελπίζετε πως στο μέλλον ο άλλος θα σας το ανταποδώσει όταν χρειαστείτε εσείς τη βοήθεια του.

·         Φοβάστε να «κλείσετε πόρτες». Ουσιαστικά φοβάστε πως μια άρνηση συνεπάγεται χαμένες ευκαιρίες.

·         Φοβάστε πως ο άλλος θα εκλάβει την άρνηση ως απόρριψη και μαζί με «κλείσιμο πόρτας» δεν θέλετε να κόψετε και γέφυρες επικοινωνίας.

Το αποτέλεσμα είναι να παίρνετε συχνά τον τίτλο «Ο υπάλληλος/φίλος/σύντροφος της χρονιάς» αλλά με κόστος απεριόριστη κούραση και εκνευρισμό γιατί νιώθετε και θύμα εκμετάλλευσης.

Γιατί οι γυναίκες λένε «ΟΧΙ» δυσκολότερα;

Είναι γεγονός ότι οι γυναίκες έχουν μεγαλύτερη δυσκολία να πουν «όχι». Μαθημένες από μικρές να αφουγκράζονται τις ανάγκες και τα συναισθήματα των γύρω τους και να παίρνουν το ρόλο του ειρηνευτή και του διαμεσολαβητή για να διασφαλίσουν την αρμονία στις σχέσεις, πολύ συχνά αναπτύσσουν την «ικανότητα» να λένε αυτόματα «ναι» σε ότι τους ζητήσουν. Ωστόσο, πολύ συχνά, αυτή τους η «ικανότητα» γίνεται με το καιρό βάρος που τις πιέζει αβάσταχτα και που τελικά, αντίθετα με τον αρχικό σκοπό, επιδρά αρνητικά τόσο στις ίδιες όσο και στις σχέσεις τους.

 

Εξασκηθείτε στην άρνηση:

·         Πείτε απλά «Συγνώμη, δεν μπορώ» .

Χρησιμοποιείστε ένα απλό, συμπαθητικό αλλά κοφτό τρόπο.

Αν ερωτηθείτε για ποιό λόγο δεν μπορείτε, απλά πείτε ότι δεν προλαβαίνετε και αλλάξτε θέμα.

Οι περισσότεροι φυσιολογικοί άνθρωποι θα είναι διακριτικοί και δεν θα δώσουν έκταση. Ωστόσο, αν βρεθείτε υπό πίεση, επαναλάβετε: «Σου εξήγησα, θα ήθελα πολύ να το κάνω αυτό για σένα αλλά δεν μπορώ».

 

·         Μη δικαιολογείστε: Για να αποφύγετε μία ευθεία άρνηση πέφτετε στην παγίδα της δικαιολογίας. Δυστυχώς οι δικαιολογίες περιπλέκουν αντί να ξεκαθαρίζουν την κατάσταση. Σας απομακρύνουν από το στόχο, προκαλούν ανεξέλεγκτες και ακατάλληλες αντιδράσεις και σας εκθέτουν.

 

·         Μην μοιράζετε ψεύτικες ελπίδες.

Αποφύγετε εκφράσεις του τύπου: «Θα δούμε/θα το συζητήσουμε/θα σε πάρω αργότερα» αν πραγματικά δεν το εννοείτε.

Αν είστε σίγουροι ότι δεν μπορείτε να πραγματοποιήσετε αυτό που σας  ζητείται, είναι προτιμότερο να πείτε ευθέως και αμέσως το όχι παρά να δημιουργήσετε προσδοκίες αυξάνοντας τη δυσαρέσκεια αργότερα.

 

·         Πείτε την αλήθεια.

Αν πραγματικά θα θέλατε να κάνετε αυτό που σας ζητάει κάποιος αλλά δεν έχετε το χρόνο, μπορείτε απλά να πείτε: «Δεν μπορώ να κάνω αυτό που μου ζητάς, αλλά μπορώ σίγουρα αυτό...», προτείνοντας μια πιο χαλαρή και εύκολα υλοποιήσιμη δέσμευση.

·         Μην υπερεκτιμάτε τις δυνάμεις σας.

Πείτε όχι σε κάτι που είναι αδύνατον να πραγματοποιήσετε. Αν υπερεκτιμήσετε τις δυνάμεις σας θεωρώντας ότι στο τέλος θα τα βγάλετε πέρα, μπορεί να μπλεχτείτε σε ένα κυκεώνα παρεξηγήσεων.

Εάν σας φορτώσουν δουλειά, μην αντιδράσετε σαν να πρόκειται για μια ρουτίνα που γνωρίζετε καλά. Ίσως οι δυσκολίες να είναι ανυπέρβλητες. Εάν αναλάβετε ακόμα μια δουλειά στο γραφείο λέγοντας μέσα σας «θα τα καταφέρω» ίσως δέχεστε να κάνετε κάτι χωρίς να λάβετε υπόψη σας την πραγματικότητα με αποτέλεσμα να  κάθεστε αργά στη δουλειά, να έχετε λιγότερο προσωπικό χρόνο και να νιώθετε ριγμένοι. Αυτό θα σας εξαγριώσει πρώτα με τον εαυτό σας και έπειτα με τους άλλους.

·         Προετοιμαστείτε.

Η προετοιμασία της άρνησης περιλαμβάνει και τον τρόπο έκφρασης. Ένα προετοιμασμένο «όχι» το οποίο έχετε προβλέψει και μελετήσει μπορεί να σας διευκολύνει. Ένα «όχι» που συγκρατείτε και στριφογυρίζετε στο μυαλό σας διαφέρει από ένα αυθόρμητο «όχι». Ένα «όχι» αδύναμο και χωρίς πειθώ ακούγεται ψεύτικο ενώ ένα «όχι» απότομο ακούγεται σαν ξέσπασμα. Ξεκινήστε με ένα «όχι αλλά...». Δείχνει ευγένεια, δεν τονίζει την άρνηση, υπογραμμίζει τη θετική εναλλακτική πρόταση και έτσι η απάντηση είναι λιγότερο απότομη και περισσότερο ανοιχτή. Αυτό ς ο τύπος άρνησης χρησιμοποιείται συχνά στις διαπραγματεύσεις ώστε να ενισχυθεί η ποιότητα της επικοινωνίας και να ενθαρρυνθεί ο διάλογος.

·         Μη ρίχνετε τις ευθύνες στους άλλους: Αντί να θυμώσετε με τον εαυτό σας που δεν μπόρεσε να πει όχι για να αποφύγει την βαρετή διάλεξη ξεσπάτε στο φίλο σας που δεν του αρνηθήκατε. Η μετάθεση των συναισθημάτων σας στους άλλους είναι μία πρακτική που ανακουφίζει πρόσκαιρα, απελευθερώνοντας την ένταση. Όμως η στρατηγική του αποδοπομπιαίου τράγου δεν καλύπτει την αδυναμία σας να πείτε όχι.

 

Να είστε απλοί και ολιγόλογοι. Μην είστε υπερβολικά απολογητικοί για να μην μπει η άλλη πλευρά στην διαδικασία να σας μεταπείσει.

Μην ξεχνάτε πως ο χρόνος σας είναι περιορισμένος και πολύτιμος  οπότε είναι σημαντικό να μάθετε πώς να τον αξιοποιείτε και να τον εκμεταλλεύεστε.

 

 

Πώς να πείτε «όχι» σε ένα συνάδελφο (που σας φορτώνει επιπλέον δουλειά).

 

Πιθανές απαντήσεις:

·         «Ευχαριστώ που με ρωτάς, αλλά έχω ήδη αρκετά να κάνω και δεν μπορώ να αναλάβω κάτι άλλο αυτή τη στιγμή».

·         «Ευχαριστώ για την προσφορά, αλλά είμαι απασχολημένος, ίσως ο τάδε θα μπορούσε να σε βοηθήσει καθώς έχει τα προσόντα που χρειάζεσαι».

 

Στην απάντηση μας πρέπει να είναι ξεκάθαρος ο φόρτος εργασίας μας, η ποιότητα της και οι προτεραιότητες ώστε αυτός που ρωτάει να μην μπορεί να το αμφισβητήσει.

 

Πώς να πείτε «όχι» στον προϊστάμενο με τις παράλογες απαιτήσεις.

Είναι γεγονός ότι το «όχι» σε έναν προϊστάμενο είναι πιο δύσκολο από το «όχι» σε ένα συνάδελφο. Ωστόσο, και σε αυτή τη περίπτωση, στην απάντηση καλό είναι να υπογραμμίζεται ο φόρτος εργασίας και γιατί όχι, να ζητείται βοήθεια.

Πιθανές απαντήσεις:

 

·        «Με κολακεύει που με θεωρείτε ικανό για αυτό αλλά έχω αρκετή εργασία και    δεν μπορώ να κάνω κάτι άλλο, εκτός αν επιθυμείτε να αλλάξω τις προτεραιότητες της τρέχουσας εργασίας μου».

·    «Εκτιμώ που μου το ζητάτε αλλά δεδομένου του φόρτου εργασίας μου, η       ποιότητα της δεν θα είναι καλή αν αναλάβω κάτι άλλο αυτή τη στιγμή».

·        «Δεν υπάρχει χώρος στο πρόγραμμα μου μέχρι το τέλος αυτού του μήνα αλλά τα πράγματα είναι εύκολα για εμένα τον Μάρτιο. Αν εξακολουθείτε να χρειάζεστε κάποια βοήθεια από εμένα, θα είμαι στην ευχάριστη θέση να συμμετάσχω».

Και αν δεν τα καταφέρετε;

Υπάρχει το ενδεχόμενο να μην πιάσει κάτι από τα παραπάνω ή να μην σας δοθεί η ευκαιρία να τα χρησιμοποιήσετε. Σε τέτοιες περιπτώσεις, μπορείτε να ζητήσετε λίγο χρόνο για να επανέλθετε με κάτι από τα παραπάνω. Επιπλέον, ζητώντας λίγο χρόνο, έχετε την ευκαιρία να συγκεντρώσετε τις πληροφορίες που χρειάζεστε για να εντοπίσετε τους λόγους για τους οποίους δεν θέλετε να κάνετε κάτι και να είστε πιο σίγουροι όταν θα αρνηθείτε.

Πιθανές απαντήσεις:

·         «Λυπάμαι αλλά θα πρέπει να ελέγξω κάποια πράγματα πριν δώσω μία απάντηση».

·         «Δεν είμαι σίγουρος ότι μπορώ να το κάνω. Μπορώ να σας απαντήσω αργότερα;»

 

Μην έχετε ενοχές όταν οι άλλοι αντιδρούν με θυμό ή εκνευρισμό. Είναι μέσα στα δικαιώματα μας να λέτε «όχι» ακόμα και χωρίς λόγο. Ένα απλό «λυπάμαι αλλά δεν μπορώ» μπορεί να είναι προτιμότερο ειδικά σε περιπτώσεις όπου αυτό που μας ζητάνε να κάνουμε έχει προθεσμία.

Για να βάλετε στην καθημερινότητα σας το «Όχι»:

1.    Σκεφτείτε πρώτα τις δικές σας ανάγκες

Υπάρχουν περιπτώσεις όπου για να μην δυσαρεστήσετε τους άλλους δέχεστε να προσφέρετε τη βοήθεια σας χωρίς να σκεφτείτε εάν είστε σε θέση.

 

2.    Μην ξαφνιάζεστε

Υπάρχει πιθανότητα να δεχτείτε κάτι επειδή η ερώτηση σας ξάφνιασε. Δεν είχατε τον απαραίτητο χρόνο να σκεφτείτε αν μπορείτε να αναλάβετε περισσότερες ευθύνες και καταλήγετε να αποφορτίζετε τους άλλους για να φορτιστείτε εσείς. Όταν έρχεστε σε δύσκολη θέση και δεν ξέρετε τί να απαντήσετε μπορείτε απλά να ζητήσετε πίστωση χρόνου για να το σκεφτείτε. Μπορείτε να ρωτήσετε: «Μπορώ να σου απαντήσω αύριο γιατί δεν θυμάμαι αυτή τη στιγμή αν έχω κανονίσει κάτι άλλο;».

 

3.    Διώξτε τις ενοχές

Κάθε φορά που αρνείστε κάτι σας κατακλύζουν οι τύψεις. Ωστόσο, δεν είναι υποχρέωση σας να είστε πάντα διαθέσιμη όταν σας χρειάζονται οι φίλοι, η οικογένεια σας ή ο σύντροφός σας. Το να αρνηθείτε κάτι σε κάποιον δεν σημαίνει ότι δεν τον υπολογίζεται. Να θυμάστε ότι αυτοί που σας αγαπούν δεν θα σας κρίνουν αρνητικά αν πραγματικά δεν μπορείτε να τους ακολουθήσετε ή να τους εξυπηρετήσετε.

 

·         Το να λέτε ΟΧΙ δεν δείχνει αγένεια και έλλειψη σεβασμού εκτός αν είναι η μόνη λέξη που λέτε. Το «ΟΧΙ» αποτελεί ένα ατομικό, απεριόριστο δικαίωμα που φανερώνει πόσο εκτιμάτε το χρόνο και τα όρια που βάζετε στους ανθρώπους γύρω σας.

·         Να θυμάστε ότι κάθε όχι σημαίνει ναι για κάτι άλλο και το αντίστροφο.
