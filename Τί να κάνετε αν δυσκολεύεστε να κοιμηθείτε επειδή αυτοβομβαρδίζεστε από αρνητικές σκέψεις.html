Τί να κάνετε αν δυσκολεύεστε να κοιμηθείτε επειδή αυτοβομβαρδίζεστε από αρνητικές σκέψεις

1.    Καταγράψτε τις πιο αγαπημένες σας αναμνήσεις (π.χ. τις καλύτερες διακοπές, το πιο εκπληκτικό κατόρθωμα, το πιο διασκεδαστικό απόγευμα ή σαββατοκύριακο, ή την πιο ευτυχισμένη στιγμή σας).

 

2.    Αν νιώθετε πολύ αποθαρρυμένος για να σκεφτείτε μια ευχάριστη ανάμνηση, καταγράψτε τα μέρη που θα θέλατε να επισκεφτείτε ή τα πράγματα που θα θέλατε να κάνετε.

 

3.    Βάλτε αυτές τις λίστες δίπλα στο κρεβάτι σας.

 

4.    Πριν σβήσετε το φως, διαλέξτε δύο πράγματα από τις λίστες σας.

 

5.    Αμέσως μόλις σβήσετε το φως και ακουμπήσετε το κεφάλι σας στο μαξιλάρι, επικεντρωθείτε στο ένα από τα πράγματα που επιλέξατε. Για παράδειγμα, αν πρόκειται για τις αγαπημένες σας διακοπές, σκεφτείτε διεξοδικά αυτές τις διακοπές: Πώς αποφασίσετε να πάτε; Πώς πήγατε εκεί; Ποιός ήταν μαζί σας; Τί κάνατε; Επικεντρωθείτε σε όλες τις ειδικές λεπτομέρειες του ταξιδιού.

 

6.    Μην επιτρέψετε σε καμία σκέψη να επηρεάσει τη συγκέντρωση σας. Αν παρέμβουν άσχετες σκέψεις, αγνοήστε τις και επιστρέψτε στις λεπτομέρειες των διακοπών σας.

 

7.    Αν σκεφτείτε όλα όσα σχετίζονται με αυτές τις υπέροχες διακοπές και εξακολουθήσετε να είστε ξύπνιος, προχωρήστε στη δεύτερη επιλογή σας.

 

8.    Προσπαθήστε να είναι όλες οι σκέψεις σας θετικές.

 

9.    Κάντε αυτή την άσκηση όσο συχνά χρειάζεται.

 

Στην αρχή μπορεί να σας φανεί δύσκολο να συγκεντρωθείτε σε αυτή την άσκηση. Είναι πιθανόν να εξακολουθήσουν να παρεμβαίνουν ανεπιθύμητες σκέψεις.  Ωστόσο, με τον καιρό, θα μπορείτε να ελέγχετε τις σκέψεις σας και αυτή η άσκηση θα σας βοηθάει να κοιμάστε εύκολα.
