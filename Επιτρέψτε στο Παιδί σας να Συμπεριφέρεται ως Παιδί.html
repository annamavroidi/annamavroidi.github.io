Επιτρέψτε στο Παιδί σας να Συμπεριφέρεται ως Παιδί

Ο σεβασμός απέναντι στο παιδί σας έχει να κάνει εν μέρει με το να του επιτρέπεται να συμπεριφέρεται ανάλογα με την ηλικία του. Αυτό σημαίνει ότι θα πρέπει να χαίρεστε το κάθε στάδιο ανάπτυξης του παιδιού σας και να αντιστέκεστε στον πειρασμό να το σπρώχνετε πρόωρα προς το επόμενο στάδιο. Αφήστε την ανάπτυξη του να ξεδιπλωθεί χωρίς να προσπαθείτε να την κατευθύνετε συνεχώς.

Είναι φυσιολογικό να θέλετε να δημιουργήσετε δομή στη ζωή του παιδιού σας ούτως ώστε να ακολουθήσει το δρόμο προς την ευτυχία και την επιτυχία. Ωστόσο, πρέπει επίσης να φροντίζετε ώστε να χαίρεται το παρόν. Επιτρέψτε στο παιδί σας να συμπεριφέρεται ως παιδί. Θα μεγαλώσει πολύ πιο γρήγορα από όσο φαντάζεστε, ακόμα και χωρίς τη δική σας βοήθεια.

Τα παιδιά μεγαλώνουν με διαφορετικούς ρυθμούς το καθένα και μπορούν να αναπτυχθούν πιο γρήγορα στον ένα τομέα (π.χ. στην κινητική ανάπτυξη) αλλά να καθυστερήσουν περισσότερο σε άλλους τομείς (π.χ. στη γλώσσα). Αφήστε το παιδί σας να αναπτυχθεί με τους δικούς του ρυθμούς. Εάν ανησυχείτε ότι αναπτύσσεται πολύ αργά κατά τη διάρκεια της βρεφικής ηλικίας ή σε οποιοδήποτε άλλο στάδιο, τότε συμβουλευτείτε τον παιδίατρο σας. Συνήθως, θα σας πει ότι δεν πρέπει να ανησυχείτε.

Επιτρέψτε στο νήπιο σας να μάθει πράγματα για τον κόσμο αφήνοντας το να ανακαλύψει μόνο του ότι υπάρχει γύρω του. Αυτό που χρειάζεται σε αυτή τη φάση είναι ένα ασφαλές περιβάλλον το οποίο να μπορεί να εξερευνήσει καθώς και γονείς που θα χαρούν να κυλιστούν στο πάτωμα και να παίξουν μαζί του.

Μην μετατρέπετε κάθε επαφή με το παιδί σας σε μάθημα. Θα μάθει πάρα πολλά πράγματα από εσάς εάν του διαβάζετε καθημερινά και εάν περνάτε αρκετό ποιοτικό χρόνο μαζί του.

Μην αναγκάζετε το παιδί προσχολικής ηλικίας να φέρεται σαν μεγάλος. Τα τετράχρονα παιδιά φωνάζουν, τα κάνουν όλα άνω-κάτω, κάνουν θόρυβο, ανοησίες και δεν δείχνουν προσοχή. Η ζωή σας ως γονιού θα είναι ευκολότερη εάν προσαρμοστείτε και φανείτε και εσείς παιχνιδιάρηδες. Όταν το παιδί σας θα μεγαλώσει, θα νιώσει την πίεση να πρέπει να φανεί σοβαρό και πειθαρχημένο οπότε, για την ώρα, αφήστε το να συμπεριφερθεί ως παιδί προσχολικής ηλικίας αντί να το εξαναγκάσετε να γίνει μικρομέγαλο.

Στο παιδί του δημοτικού σχολείου φροντίστε να αφήνετε χρόνο για παιχνίδι. Προσπαθήστε να βρείτε την ισορροπία ανάμεσα στις προγραμματισμένες και τις ελεύθερες δραστηριότητες. Η ζωή του εννιάχρονου παιδιού δεν πρέπει να είναι απλώς ένα ατέλειωτο σχολείο.

Στο έφηβο παιδί σας επιτρέψτε να διασκεδάσει την εφηβεία του. Είναι ωραίο να το βοηθήσετε να ωριμάσει σταδιακά ώστε να φτάσει στην ενηλικίωση. Το να επιμένετε να συμπεριφέρεται σαν ενήλικος όταν είναι μόλις δεκατριών χρόνων δεν είναι καθόλου καλό. Είναι φυσιολογικό οι έφηβοι να κάνουν τρέλες πότε πότε, να υπερβάλλουν, να πειραματίζονται και να ζουν έντονα την κάθε στιγμή. Η δουλειά σας ως γονιού δεν είναι να καταπνίξετε όλες αυτές τις τάσεις αλλά να κάνετε το καλύτερο δυνατό για να φροντίσετε να μην πάθει κάτι κακό το παιδί σας στην πορεία.
