Για καλύτερη επικοινωνία με τους εφήβους

  • Δεν θα πρέπει να προσφέρετε συμβουλές στους εφήβους όταν εκείνοι δεν σας τις έχουν ζητήσει. Ακόμα και αν ζητήσουν την άποψη σας, μην την εκφράσετε προτού ζητήσετε την δική τους άποψη.
    ΓΙΑΤΙ; Αυτό πρέπει να γίνεται επειδή οι έφηβοι έχουν μόλις φτάσει στην ηλικία όπου μπορούν να χρησιμοποιούν την λογική τους και διαμορφώνουν δικές τους απόψεις. Είναι ικανοί να λάβουν υπόψη τους και τις γνώσεις τρίτων, χρειάζονται όμως και κάποιον να ακούει με συμπάθεια και να υπολογίζει τις δικές τους απόψεις.

  • Να διαθέτετε χρόνο για συζητήσεις με τους εφήβους πάνω σε θέματα πέραν εκείνων που επιθυμείται να κάνουν. Για παράδειγμα, συζητήστε μαζί τους για τα θέματα που μελετούν στην ιστορία και ακούστε τις απόψεις τους.
    ΓΙΑΤΙ; Σε αυτή τη φάση, οι έφηβοι έχουν ανάγκη να διαφωνούν και να εκφράζουν τις δικές τους απόψεις. Επιπλέον, με αυτό το τρόπο θα μειώσετε και τα αίτια των αντιδράσεων τους.

  • Ακόμα και όταν δεν συμφωνείτε με κάποια άποψη τους, οφείλετε να εκτιμήσετε, τουλάχιστον, το συλλογισμό τους. Δώστε τους να καταλάβουν ότι έχετε ανοιχτό μυαλό για οποιοδήποτε άλλο θέμα, πέρα π.χ. από τη διορία των νυχτερινών εξόδων τους.
    ΓΙΑΤΙ; Οι έφηβοι νιώθουν την ανάγκη να διαφοροποιηθούν. Κατά συνέπεια, θα είναι μια σημαντική εμπειρία για εκείνους να μάθουν ότι έχουν δικαίωμα να διαφωνούν και να έχουν διαφορετικές απόψεις, εφόσον τους δείχνετε ότι εκτιμάτε τις απόψεις και τη λογική τους.
    Αν είστε δεκτικοί στις απόψεις τους, δεν θα είναι τόσο απαιτητικοί όταν επιδιώξουν περισσότερες ελευθερίες από όσες είναι έτοιμοι να δεχτούν. Αν δεν τους δώσετε ευκαιρίες να διαφωνούν για τρέχοντα θέματα, τότε θα νιώθουν την ανάγκη να διαφωνούν μαζί σας σε προσωπικό επίπεδο.

  • Μην συμπεριφέρεσθε σαν δικτάτορες! Οι έφηβοι σίγουρα θα αντιδράσουν.
    ΓΙΑΤΙ; Είναι σημαντικό να γνωρίζετε ότι οι έφηβοι έχουν πολύ πιο αναπτυγμένη την αίσθηση του δικαίου από τα παιδιά. Κατά συνέπεια, με το να τους ακούτε και να συνεργάζεστε μαζί τους στο να καθορίσετε πόσες ελευθερίες θα πρέπει να τους παραχωρηθούν, θα ενισχύσετε τη σχέση μεταξύ σας.

  • Όταν ένας έφηβος αντιδρά στα αιτήματα σας, αντί να του κάνετε κήρυγμα, ανοίξτε συζήτηση μαζί του. Για παράδειγμα, ρωτήστε τον : « Γιατί νομίζεις ότι θέλω να κάνεις αυτό;» Προσοχή! Δεν είναι σωστό και έντιμο να ρωτάτε: « Έχεις ιδέα πώς με κάνει να νιώθω αυτό;». Είναι σημαντικό να θυμάστε ότι τα παιδιά, μέχρι την ηλικία των δεκαοχτώ ετών, στηρίζονται στους γονείς τους, ωστόσο, δεν είναι υπεύθυνα για τα συναισθήματα των γονιών τους.

Να θυμάστε:

  • Προτού δώσετε μία εντολή στον έφηβο, θα πρέπει πρώτα να ζητήσετε τη συνεργασία του, να δίνετε προσοχή στις πηγές της αντίδρασης του και να σέβεστε τις απόψεις του. Μόνο τότε θα πρέπει να εκφράσετε την επιθυμία σας.
    Παράδειγμα «Κατανοώ ότι πιστεύεις πως δεν είναι δίκαιο. Θέλεις να περάσεις περισσότερο χρόνο με τους φίλους σου ενώ εγώ σου ζητώ να μείνεις εδώ και να δεις τους θείους σου. Ξέρω ότι δεν το θέλεις, αλλά είναι σημαντικό για μένα. Θέλω να είσαι εδώ. Θέλω να είσαι φιλικός και ευγενικός με τους θείους σου για δύο ώρες και ύστερα θα είσαι ελεύθερος να φύγεις».

  • Όταν κοπούν οι γέφυρες επικοινωνίας, ο έφηβος σας κινδυνεύει να επηρεαστεί από τους συνομηλίκους του, οι οποίοι μπορεί να βρίσκονται εκτός ελέγχου ή ασκούν κακή επιρροή πάνω του (Gray, 2004).
