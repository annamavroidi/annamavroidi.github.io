Η διάγνωση του καρκίνου

Η διάγνωση του καρκίνου είναι δυνατόν να προκαλέσει μια σειρά από έντονα συναισθήματα όπως:

·         Φόβο για πιθανό θάνατο

·         Οργή

·         Ελπίδα

·         Ενοχή

·         Άρνηση

·         Θλίψη

·         Μοναξιά ή

·         Προσμονή.

 

Τα παραπάνω συναισθήματα είναι φυσιολογικές αντιδράσεις σε μια τόσο σημαντική αλλαγή.

Η διάγνωση του καρκίνου θα επηρεάσει τις σχέσεις με την οικογένεια, τους φίλους και τους ανθρώπους που φροντίζουν τον ασθενή.

Συχνά το πρώτο πράγμα που ρωτούν οι ασθενείς όταν μαθαίνουν ότι έχουν καρκίνο είναι: «Θα πεθάνω;» Οι ασθενείς καλό είναι να μιλήσουν με το γιατρό τους για την σημασία της διάγνωσης για εκείνους και τι επιφυλάσσει το μέλλον. Γνωρίζοντας περισσότερα για την ασθένεια τους είναι δυνατόν να περιορίσουν τον φόβο. Επίσης, ότι σκέφτονται σχετικά με την ασθένεια ίσως αλλάξει καθώς συνεχίζεται η θεραπεία.

 

Μοιραστείτε την Είδηση της Διάγνωσης

Ίσως  είναι δύσκολο για κάποιους ασθενείς που πάσχουν από καρκίνο να μοιραστουν την είδηση της διάγνωσης. Μπορεί να αισθάνονται άβολα να μιλήσουν για προσωπικά θέματα ή να μην είναι σίγουροι για το πως θα αντιδράσουν η οικογένεια και οι φίλοι ή να θέλουν να προστατεύσουν τους αγαπημένους τους. Ωστόσο, η είδηση της ασθένειας ίσως τους φέρει πιο κοντά σε αυτά τα πρόσωπα. Εάν μοιραστούν τον άγχος και το φόβο τους, θα νιώσουν πιο δυνατοί και αυτό θα τους βοηθήσει σε δύσκολες στιγμές. Δεν θα πρέπει να αντιμετωπίσουν μόνοι τον καρκίνο. Οι κοντινοί άνθρωποι των ασθενών θα θέλουν να γνωρίζουν τι συμβαίνει ώστε να μπορούν να τους προσφέρουν υποστήριξη.

Μερικές φορές οι ασθενείς πιστεύουν ότι κανείς δεν καταλαβαίνει τί περνούν. Ωστόσο, όταν χρειάζονται υποστήριξη, πρέπει να προσπαθούν να μην αποκλείουν τους άλλους. Να μιλούν για την ασθένεια τους όταν νιώθουν έτοιμοι για ερωτήσεις αλλά να βάλουν όρια (δεν είναι ανάγκη να μοιράζονται όλες τις λεπτομέρειες με όλους).

Η προσπάθεια κάποιων ασθενών να κρύψουν την κατάσταση τους είναι συνήθως ανεπιτυχής. Αργά ή γρήγορα η οικογένεια και οι φίλοι θα μάθουν ότι πάσχουν από καρκίνο. Οι περισσότεροι θα αντιληφθούν ότι κάτι τους ενοχλεί ή θα παρατηρήσουν αλλαγές στη συμπεριφορά ή την εμφάνιση τους. Η οικογένεια τους και οι κοντινοί τους φίλοι πιθανόν να πληγωθούν αν δεν τους το πουν.

Κάποιες φορές τα μέλη της οικογένειας μαθαίνουν για τον καρκίνο πριν από τον ασθενή. Μπορεί να σκεφτούν ότι ο ασθενής είναι πολύ νέος ή πολύ γέρος για να του πουν την αλήθεια.

Οι ασθενείς που πάσχουν από καρκίνο συνήθως λένε ότι θέλουν να γνωρίζουν τί συμβαίνει ώστε να μπορούν να αποφασίσουν ποιά θεραπευτική αγωγή να ακολουθήσουν ή αν οι προοπτικές θεραπείας είναι κακές, ποιός είναι ο καλύτερος τρόπος να ζήσουν τον χρόνο που απομένει.

 

  Η Επικοινωνία της Είδησης του Καρκίνου στα Παιδιά των Ασθενών

 

Κάποιο ασθενείς αποφεύγουν να πουν στα παιδιά τους ότι έχουν καρκίνο. Ωστόσο, τα μικρά παιδιά συνήθως αντιλαμβάνονται ότι υπάρχει κάποιο πρόβλημα ακόμα και αν δεν ξέρουν τι είναι. Όταν δεν τους πουν τί συμβαίνει, τα παιδιά μπορεί να φανταστούν τα χειρότερα. Επίσης, μπορεί να το μάθουν από άλλο άτομο και αυτό ίσως τους προκαλέσει θυμό και σύγχυση.

Μερικοί γονείς πιστεύουν ότι προστατεύουν τα παιδιά τους όταν τους αποκρύπτουν τα κακά νέα. Στην πραγματικότητα τα παιδιά τους θα ωφεληθούν από την ανοικτή και ειλικρινή προσέγγιση τους. Πριν από τη συζήτηση πρέπει να σκεφτούν τί θα πουν. Να μιλήσουν με τα παιδιά τους στη γλώσσα που καταλαβαίνουν, με απλές εξηγήσεις στα μικρά παιδιά. Να τα καθησυχάσουν λέγοντας τους ότι η ασθένεια τους δεν είναι δικό τους σφάλμα και να τα ενθαρρύνουν να τους πουν τί γνωρίζουν για το καρκίνο απαντώντας στις ερωτήσεις τους με ειλικρίνεια.

 

Η Ζωή Μετά τον Καρκίνο

 

Υπάρχουν πολλά διαφορετικά πράγματα που μπορεί ένα άτομο που πάσχει από καρκίνο να κάνει για να φροντίσει τον εαυτό του μετά τη θεραπεία του όπως είναι να ακολουθήσει υγιεινή διατροφή και φυσικό τρόπο ζωής, να συνεχίσει τις προηγούμενες δραστηριότητες του όπως τα χόμπι, τα σπορ ή την εργασία, να μάθει τεχνικές χαλάρωσης και να βρει χρόνο για διασκέδαση και γέλιο. Η συμβουλευτική μπορεί επίσης να το βοηθήσει να προσαρμοστεί στη νέα ζωή του. Ο καρκίνος δεν είναι πάντα μια αρνητική εμπειρία. Ασθενείς που έχουν επιβιώσει από καρκίνο λένε ότι η ασθένεια είχε θετικές επιπτώσεις στη ζωή τους.

 

Η εργασία είναι σημαντικό κομμάτι της ζωής για πολλούς ανθρώπους, όχι απλώς ένας τρόπος να εξασφαλίζουν εισόδημα. Προσφέρει ικανοποίηση και ευκαιρίες για κοινωνικότητα. Η επιστροφή στην εργασία είναι ένας τρόπος να γίνει η ζωή του ατόμου που έπασχε από καρκίνο ξανά φυσιολογική.

Το ίδιο το άτομο που έπασχε από καρκίνο γνωρίζει καλύτερα πότε  

να επιστρέψει στην εργασία του. Αυτό διαφέρει για το καθένα και εξαρτάται από το πώς αισθάνεται. Μπορεί να το βοηθήσει να επιστρέψει για λίγο και να συνεχίσει όταν νιώσει καλύτερα. Είναι καλό να προσπαθεί να κάνει μικρά, προσεκτικά βήματα προόδου ώστε να μην κουράζετε πολύ. Μπορεί να εξετάσει το ενδεχόμενο να συζητήσει με τον εργοδότη του για μερική απασχόληση, εκ περιτροπής εργασία ή εργασία από το σπίτι.

Όταν το άτομο που έπασχε από καρκίνο επιστρέψει στην εργασία του, είναι πιθανόν να διαπιστώσει ότι οι σχέσεις με τους συναδέλφους του άλλαξαν. Όπως η οικογένεια και οι φίλοι του, οι συνάδελφοι του ίσως να μην ξέρουν τί να του πουν ή να προσπαθούν να μην το στεναχωρήσουν.

Καρκινοπαθείς έχουν διαπιστώσει πως διευκολύνονται οι σχέσεις με τους συναδέλφους τους όταν είναι ανοιχτοί σχετικά με την κατάσταση τους.

 

Όταν κάποιος προσβληθεί από καρκίνο δεν σημαίνει ότι πρέπει να χάσει κάθε ελπίδα. Οι προοπτικές θεραπείας για πολλές μορφές καρκίνου βελτιώνονται συνεχώς. Κάποιες μορφές είναι δυνατόν να θεραπευτούν ενώ άλλες μπορούν να ελεγχθούν. Αν ο καρκίνος δεν μπορεί να ελεγχθεί, τα συμπτώματα είναι δυνατόν να γίνουν ηπιότερα για να κάνουν τη ζωή πιο άνετη.

 

Οι συναισθηματικές επιδράσεις του καρκίνου μπορεί να κρατήσουν για μεγάλο χρονικό διάστημα μετά τη θεραπεία. Είναι συνηθισμένο για τους ασθενείς να αισθάνονται ανήσυχοι για ασήμαντα σημάδια της ασθένειας ή του πόνου, φοβούμενοι ότι θα επανέλθει η νόσος.

 

Για πολλούς ασθενείς ο καρκίνος είναι μια εμπειρία που τους αλλάζει τη ζωή. Όταν ολοκληρωθεί η θεραπεία, το άτομο που έπασχε από καρκίνο πιθανόν να περιμένει ότι η ζωή του θα γίνει ξανά όπως ήταν πριν την ασθένεια. Η προσαρμογή στη νέα κατάσταση μετά τη θεραπεία μπορεί να πάρει χρόνο. Ότι ήταν φυσιολογικό πριν, ίσως να μην είναι πλέον. Η εμπειρία του καρκίνου είναι δυνατόν να το κάνει να σκεφτεί σε βάθος τί είναι σημαντικό για εκείνο, διαμορφώνοντας μια νέα αντίληψη για τις αξίες και τις προτεραιότητες στη ζωή του.

 

Ενώ δίνει χρόνο στον εαυτό του για να προσαρμόσει τη ζωή του μετά το καρκίνο, το άτομο που έπασχε από αυτή τη νόσο πρέπει να λάβει υπ’ όψιν του ότι η οικογένεια και οι φίλοι του επίσης θα χρειαστούν χρόνο. Όπως και εκείνο, και αυτοί έχουν περάσει δύσκολες ώρες και μπορεί να χρειάζονται ένα χρονικό διάστημα ανάπαυσης και αναπροσαρμογής.
