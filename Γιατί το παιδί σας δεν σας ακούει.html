Γιατί το παιδί σας δεν σας ακούει


Αναρωτιέστε γιατί το παιδί σας δεν είναι συνεργάσιμο; Γιατί, για παράδειγμα, σας μιλάει άσχημα ή σας ζητάει επίμονα να του αγοράσετε κάτι παρά την άρνηση σας; Η απάντηση είναι ότι το παιδί σας επιδιώκει έναν από τους παρακάτω στόχους:
  • Να αποσπάσει υπερβολική προσοχή: Το παιδί νιώθει σημαντικό όταν γίνεται το επίκεντρο της προσοχής ακόμα και αν προσοχή σημαίνει ότι κάποιος έχει γίνει έξαλλος μαζί του. Σε αυτή τη περίπτωση θα πρέπει να του δώσετε προσοχή όταν συμπεριφέρεται με τρόπο κοινωνικά αποδεκτό και να το αγνοήσετε όταν δεν συμπεριφέρεται έτσι.

  • Να κάνει επίδειξη δύναμης: Το παιδί νιώθει σημαντικό όταν μπορεί να αγνοήσει μια εντολή σας ή όταν καταφέρει να ενδώσετε σε κάποια δική του. Εσείς θα πρέπει να μετατρέψετε τις ηγετικές του τάσεις σε επιθυμία να βοηθήσει την οικογένεια. Για παράδειγμα, ζητήστε την γνώμη του για κάτι ή προτρέψτε το να αποφασίσει εκείνο για κάποια μικρότερης σημασίας ζητήματα (π.χ. να επιλέξει εκείνο κάποιο ρούχο που θα του αγοράσετε).

  • Να εκδικηθεί: Το παιδί έχει την αίσθηση πως δεν το αγαπούν οι γονείς του και θέλει να τους πληγώσει. Αυτό μπορεί να συμβαίνει όταν οι γονείς καταφεύγουν στην τιμωρία για να υπερισχύσει η δική τους θέση. Σε αυτή τη περίπτωση, θα πρέπει να δίνετε στο παιδί σας αμέριστη αγάπη και στοργή αντί για τιμωρία ή λεκτικές επιθέσεις.

  • Να εκφράσει την αίσθηση της ανεπάρκειας: Το παιδί πιστεύει ότι δεν κάνει σωστά κάποια πράγματα οπότε δεν βλέπει γιατί να συνεχίσει να τα κάνει. Αν έχετε παρατηρήσει ότι έχετε πάψει να το νουθετείτε και ότι δεν περιμένετε τίποτα από το παιδί σας, τότε το παιδί σας επιδιώκει να εκφράσει ένα αίσθημα ανεπάρκειας. Σε αυτή την περίπτωση, θα πρέπει να το ενθαρρύνετε ώστε να βελτιωθεί η αυτοεκτίμηση του.

Προσοχή!

  • Να αφήνετε το παιδί σας να βιώνει τις συνέπειες της μη αποδεκτής συμπεριφοράς του (π.χ. αν δεν ετοιμαστείς σε πέντε λεπτά, δεν θα σε πάρω μαζί μου).
    Για να είναι αποτελεσματική η χρήση των συνεπειών , θα πρέπει να τηρείτε κάποιες προϋποθέσεις:
        1. Οι συνέπειες να έχουν σχέση με την συμπεριφορά.
        2. Να προσφέρετε δυνατότητες επιλογής όταν είναι δυνατόν.
        3. Το παιδί θα πρέπει να γνωρίζει ότι έχει μια δεύτερη ευκαιρία για να βελτιωθεί.
        4. Η στάση σας θα πρέπει να είναι διδακτική και ο τόνος της φωνής σας φιλικός.

  • Να συμπεριφέρεστε στα παιδιά σας με τον ίδιο σεβασμό που θα θέλατε να δείχνουν σε εσάς οι άλλοι. Η έλλειψη σεβασμού παρατηρείται από γονείς που επιβάλλουν στο παιδί τους την θέληση τους, συχνά με τρόπο ταπεινωτικό.

  • Να ενθαρρύνετε το παιδί σας.
    Η ενθάρρυνση κάνει τα παιδιά να νιώθουν όμορφα με τον εαυτό τους ακόμα και όταν έχουν κάνει λάθος. Αντιθέτως, όταν τα αποθαρρύνετε, νιώθουν πως ότι και αν κάνουν, δεν θα είναι ποτέ αρκετό. Μπορείτε να επιτύχετε να ενθαρρύνετε τα παιδιά σας αν διαχωρίσετε την ανάρμοστη συμπεριφορά από το πρόσωπο που έχει συμπεριφερθεί ανάρμοστα. Για παράδειγμα, αν το παιδί σας ρίξει ένα ποτήρι γάλα στο πάτωμα, μην του πείτε ότι είναι απρόσεκτο. Πείτε:'' Έχυσες το γάλα σου. Έλα να καθαρίσουμε το πάτωμα μαζί και θα πιείς άλλο ''. Με αυτό το τρόπο η απρόσεκτη συμπεριφορά μετατρέπεται σε εμπειρία από την οποία το παιδί σας θα αποκομίσει κάτι.

    • (Σαπίρο και συν., 1997)
